    Data Handling

    dataHandling

    This module holds data handling capabilities for metadatasets.

    Concretely, the UI has a globally accessible data handler of type DataHandling that will take care of the datasets held by the GUI.

    DataHandling

    This class handles data.

    It checks for availability in the filesystem, and if not, creates the data structure. It also takes care for the storage on disk. The data are stored as a pickle file.

    The class should not be called as a static. Rather, there should at any given time be an instance of this class (data_handler) be available. All calls should be done on this instance, as it holds the actual data representation.

    refresh_ui(self)

    Refresh all values in the UI according to the saved values.

    This method also invokes on-the-fly validation. Note: Calling this method discards all changes that have not been updated in the metaDataSet.

    update_all(self)

    Update data according to the values currently in the GUI.
