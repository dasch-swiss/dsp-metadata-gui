    Function add_project()

    add_project(self, folder_path, shortcode, files)

    Add a new project.

    This project adds a new project folder to the collection after the user specified the folder.

    The Project is appended at the end of the list.

    Parameters:

    Name Type Description Default
    folder_path str

    path to the project folder

    		 required
    shortcode str

    the project shortcode

    		 required
    files list

    the files in the project folder

    		 required

    Function validate_graph()

    validate_graph(self, dataset)

    Validates all properties in a specific MetaDataSet.

    Does not validate each of the properties separately, but rather generates the RDF graph, which then gets validated.

    Function update_all()

    update_all(self)

    Update date from GUI.

    Calling this function iterates over each Property in the dataset and updates it with the value found in its corresponding GUI component.

    Function refresh_ui()

    refresh_ui(self)

    Refresh all values in the UI according to the saved values.

    This method also invokes on-the-fly validation. Note: Calling this method discards all changes that have not been updated in the metaDataSet.
